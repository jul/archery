What is archery? 
================

It is an enhancement of MutableMapping based on Mixins. It currently only offers:

* addition;
* substraction;
* multiplication;
* division. 

And also mixins for:

* searching

Or treating dict as vectors giving them:

* norms (abs)
* cosine
* dot product
